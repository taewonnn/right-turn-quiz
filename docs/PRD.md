# PRD: Right Turn Quiz

## 목표
우회전 단속 강화로 헷갈리는 운전자들이 퀴즈를 통해 법규를 재미있게 익히고, 결과 공유로 바이럴 유도

## 사용자
운전면허 보유 성인, 특히 최근 우회전 단속 강화로 "나 혹시 위반한 적 있나?" 불안한 운전자

## 핵심 기능
1. 우회전 상황별 퀴즈 10문항 (O/X 또는 3지선다)
2. 문항별 즉각 정오답 피드백 + 간단한 해설
3. 최종 결과 화면 — 점수 + 등급 + 팩폭 멘트 + 공유 버튼

## MVP 제외 사항
- 로그인 / 회원가입
- 결과 히스토리 저장
- 문항 관리 어드민 CMS
- 랭킹 / 통계 기능
- 다크모드

## 디자인
- 토스 스타일 — React Native 네이티브 컴포넌트 기반 미니멀
- 화이트 배경 + 포인트 컬러 1가지 (Blue #3182F6)
- Pretendard 폰트 (한글 최적화)
- 문항 전환 슬라이드 애니메이션 (Reanimated)
